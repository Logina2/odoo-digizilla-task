# Digizilla Odoo Module (Odoo v17 Community)

## Overview

Digizilla is a custom Odoo v17 Community addon developed as a technical assessment task. The module provides a complete CRUD solution with multiple views, message tracking, JavaScript customization, and QWeb PDF reporting.

## Key Features

* Custom Odoo model with tracked fields
* Tree, Form, and Kanban views
* Message chatter and change logging
* Create button hidden in Form View using JavaScript
* Dynamic PDF report generation using QWeb

## Technical Stack

* Odoo 17 Community
* Python (Odoo ORM)
* XML (Views & QWeb)
* JavaScript (UI customization)

## Installation

1. Add the `digizilla` module to your Odoo addons directory
2. Restart the Odoo server
3. Update the Apps list
4. Install **Digizilla Management**

## Notes

The PDF report is generated dynamically from Odoo via the Print menu and excludes message chatter as required.
